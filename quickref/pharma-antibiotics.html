🧠 Memory Aid: β-Lactamase Inhibitors

"CAST" — Clavulanate (with amoxicillin), Avibactam (with ceftazidime), Sulbactam (with ampicillin), Tazobactam (with piperacillin)

🧠 Memory Aid: Aminoglycoside Toxicity

Aminoglycosides are nephrotoxic and ototoxic — check trough levels, monitor creatinine, and ask about hearing.

💎 Stewardship Pearl: Broader is not better. Start narrow when possible, culture before treating, and de-escalate based on sensitivities. Reserve carbapenems and vancomycin for confirmed resistant organisms.
