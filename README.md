## Getting Started

1. Open `index.html` in any modern web browser
2. Enter your name (for progress tracking)
3. Choose a training mode:
   - **Learning Mode**: Detailed feedback at each step
   - **Clinical Mode**: Realistic presentations, minimal hand-holding
   - **OSCE Simulation**: Timed 10-minute encounters

No installation or server required - runs entirely in the browser.

## Technical Details

- **Single-file application**: All HTML, CSS, and JavaScript in one file
- **No dependencies**: Uses Tailwind CSS via CDN
- **Local storage**: Progress saved in browser
- **Responsive design**: Works on desktop and tablet
- **Dark mode**: Respects system preferences
