# ReasonDx

A comprehensive clinical reasoning trainer for medical students. Practice diagnostic thinking through realistic patient scenarios with gamification, spaced repetition, and evidence-based feedback.

## Features

### ü©∫ Clinical Case Simulator
Work through **200+ patient cases** across multiple specialties:
- Build differential diagnoses from chief complaints
- Categorize diagnoses as "Most Likely," "Must Not Miss," and "Less Likely"
- Gather targeted history using the OLDCARTS framework
- Perform focused physical examinations
- Order and interpret diagnostic workup (labs, imaging, procedures)
- Receive detailed feedback with teaching points and key findings

### üìû Consult Trainer (SBAR Practice)
Practice calling consults with **13 realistic scenarios**:
- Review patient data and identify key findings
- Construct SBAR presentations (Situation, Background, Assessment, Recommendation)
- Compare your presentation against model answers
- Covers ED, floor, and ICU settings

## Getting Started

1. Open `index.html` in any modern web browser
2. Enter your name (for progress tracking)
3. Choose a training mode
4. Start with the **Case of the Day** or browse by specialty

**No installation required** - runs entirely in the browser.

## Technical Details

- **Single-file application**: All HTML, CSS, and JavaScript in one file (~80,000 lines)
- **No backend required**: Uses browser local storage for progress
- **Responsive design**: Works on desktop and tablet
- **Dark mode**: Toggle or follows system preferences
- **Offline capable**: Works without internet after initial load

## For Educators

ReasonDx supports classroom use:
- **Live Sessions**: Create sessions with join codes for real-time participation
- **Group Discussion**: Project cases for collaborative learning
- **Progress Tracking**: Monitor student performance across cases
- **Custom Pacing**: Control case progression in guided sessions
