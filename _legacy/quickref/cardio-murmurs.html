🧠 Memory Aid: Murmurs that INCREASE with Valsalva

Only 2: HOCM and MVP (both decrease LV volume → worsen obstruction/prolapse). Everything else gets quieter.

💎 Clinical Pearl: Diastolic murmurs are ALWAYS pathologic. Systolic murmurs can be innocent (flow murmurs), especially in children and pregnancy.

🧠 Memory Aid: Maneuver Logic

Squatting = ↑ preload + ↑ afterload → louder for most murmurs, quieter for HOCM/MVP.
Valsalva/Standing = ↓ preload → quieter for most, louder for HOCM/MVP.
Hand grip = ↑ afterload → louder for regurgitant lesions (MR, AR, VSD).
