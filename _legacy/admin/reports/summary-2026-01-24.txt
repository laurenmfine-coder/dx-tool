🔧 SETTING UP AUTOMATED AUDITS
───────────────────────────────────────────────────────────────

   Option 1: Cron Job (Linux/Mac server)
   ───────────────────────────────────────
   Add to crontab (crontab -e):

   # Weekly link check (Mondays at 6 AM)
   0 6 * * 1 cd /path/to/DxSuite && node admin/audit-engine.js --links

   # Monthly full audit (1st of month at 6 AM)
   0 6 1 * * cd /path/to/DxSuite && node admin/audit-engine.js --full

   Option 2: GitHub Actions (if hosted on GitHub)
   ───────────────────────────────────────
   Create .github/workflows/audit.yml
   See admin/github-action-template.yml for example

   Option 3: Manual Reminder
   ───────────────────────────────────────
   Set calendar reminders to run audits manually
