Generate codes to give participants after they complete your Microsoft Forms consent

📋 How This Works

Microsoft Forms + Consent Codes Flow

  1. Create consent form in Microsoft Forms with all required IRB language
  2. Generate codes using this tool
  3. Add codes to Forms as the completion message (see options below)
  4. Participant completes form → sees their consent code
  5. Participant enters code on DxSuite registration page
  6. Code verified → can proceed with registration

Three Options for Distributing Codes:

Option How It Works Security Setup
Recommended Single Code Everyone gets the same code (e.g., CONSENT-DXSUITE-2026) Low (code can be shared) Easiest
Rotating Codes Change the code monthly (CONSENT-JAN26, CONSENT-FEB26) Medium Easy
Most Secure Unique Codes Each participant gets a unique code via email after submitting High Requires Power Automate

🔧 Microsoft Forms Setup

Option 1: Single Code (Simplest)

  1. Create your consent form in Microsoft Forms
  2. Go to Settings (⚙️) → Completion message
  3. Paste the "Forms Message" text generated above
  4. Save

Option 2: Unique Codes via Power Automate (Most Secure)

  1. Generate unique codes using this tool
  2. Upload codes to a SharePoint list or Excel file
  3. Create Power Automate flow: "When a new response is submitted"
  4. Flow assigns next unused code and emails it to participant
  5. Mark code as used in your list

Benefit: Each code can only be used once, preventing sharing.
