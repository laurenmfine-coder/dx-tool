ℹ️ How the Audit System Works

Automated Monitoring

The audit system tracks guideline updates from major medical organizations (AHA, ACC, AAP, IDSA, SCCM, AAAAI, ACAAI, CHEST). It uses PubMed queries to detect new publications tagged as "guidelines" or "practice parameters."

Running Audits

Manual: Click "Run Full Platform Audit" above or run node admin/audit-engine.js --full from command line.
Scheduled: Set up a cron job or GitHub Action to run weekly/monthly.

What Gets Checked

  • All reference URLs (broken link detection)
  • Content freshness (flags items >12 months old)
  • Guideline publication dates vs. DxSuite content dates
  • Internal cross-references between modules

Recommended Schedule

  • Weekly: Link verification (catches broken URLs quickly)
  • Monthly: Guideline monitoring for high-frequency sources (AHA, SCCM, IDSA)
  • Quarterly: Full content audit + less frequent guideline sources
