Meta-reasoning

The supervisory agent that
watches all the others.

Synthesizes the bus on every turn into a theory of the student's dominant error mode. Not rule-based: each mode is scored across all six memory layers simultaneously, with longitudinal pattern detection weighted 3× over single-session signals.

⚓
Anchoring Bias
Locks onto initial hypothesis despite contradicting evidence. Triggered by pattern_tags + episodic anchoring + ERS flag.
Score 7 · Confidence 0.82 · High urgency
🚪
Premature Closure
Stops revising differential before fully evaluating all evidence. Triggered by top-3 DDx unchanged post-labs.
Croskerry 2009 · Graber 2005
🔍
History Gap
Misses critical contextual history (occupation, environment, social). Triggered by missed_pivot_rate + closed_first pattern.
Graber 2005 — faulty context generation
🎯
Narrow Opener
Initial DDx has fewer than 2 diagnoses — insufficient breadth at case opening. Elstein (1978): hypothesis set size.
Elstein 1978 hypothesis set theory
🤷
Underconfident
Accurate but systematically underestimates own reasoning. Likert <3 despite >70% accuracy across sessions.
Berner & Graber 2008 · Eva & Regehr 2005
🔥
Cognitive Overload
Working memory saturated — load z >1 AND trajectory escalating. Overrides all other modes (score = 99). Immediate intervention.
Sweller 1988 · Moulton 2007
😤
Affect Disruption
Negative activated affect during history predicts downstream anchoring. ERS risk flag: valence <−0.3 + arousal >0.6 in Phase 3.
Croskerry et al. 2010 · Lerner 2015
✓
Calibrated
No dominant error mode. Student is performing at or above expected level. Response: challenge upward — press for mechanistic reasoning.
Eva & Regehr 2005 calibration theory
Three output channels per synthesis
Attending tactic
Appended to Dr. Patel's system prompt every turn. Mode-specific: "What single finding would make you reconsider?" Injected silently — the student never sees it.
Case selection
recommendNextCase() delegates to MetaReasoning.selectCase(). Anchoring mode → case with hasAnchoringTrap property. Overload mode → reduce to guided difficulty.
Difficulty calibration
Implements Vygotsky (1978) ZPD targeting. High confidence + CALIBRATED → advance difficulty. Cognitive overload → step down. Kapur (2016) challenge-first sequencing.
