Why Lactate Rises in Sepsis

Type A: Hypoperfusion

↓ O2 delivery → Anaerobic metabolism → Pyruvate → Lactate

Type B: Metabolic

Catecholamines → ↑ Glycolysis → Excess pyruvate → Lactate

Impaired Clearance

Liver dysfunction → ↓ Lactate metabolism → Accumulation

Target: Lactate clearance ≥10% in first 6 hours is associated with improved survival.

Layer 4: Treatment Targets

Mechanism-based approach to sepsis management

Hour-1 Bundle: Mechanism Rationale

1

Measure Lactate

Quantifies tissue hypoperfusion. Guides resuscitation adequacy.

Mechanism: Anaerobic metabolism marker

2

Blood Cultures Before Antibiotics

Identify pathogen for targeted therapy. Don't delay abx for cultures.

Mechanism: PAMPs → PRR → Cytokine storm (stop the source)

3

Broad-Spectrum Antibiotics

Each hour delay = ↑ mortality. Cover likely pathogens.

Mechanism: Kill bacteria → ↓ PAMP release → ↓ Inflammation

4

Crystalloid 30 mL/kg for Hypotension

Address hypovolemia from vasodilation + capillary leak.

Mechanism: ↑ Preload → ↑ CO → ↑ O2 delivery

5

Vasopressors if MAP < 65 After Fluids

Norepinephrine first-line. Target MAP ≥ 65 mmHg.

Mechanism: α1 agonist → Vasoconstriction → ↑ SVR → ↑ MAP

Vasopressor Mechanisms

Drug Primary Effect Mechanism When to Use
Norepinephrine ↑ SVR (primary), ↑ CO (mild) α1 >> β1 First-line for septic shock
Vasopressin ↑ SVR V1 receptor → Vasoconstriction Add if NE insufficient (catecholamine-sparing)
Epinephrine ↑ CO, ↑ SVR β1 + α1 Cardiogenic component suspected
Dobutamine ↑ CO β1 >> β2 Persistent hypoperfusion despite adequate MAP
