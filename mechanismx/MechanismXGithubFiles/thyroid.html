Hypothyroidism

↓ Metabolic rate → Everything slows down

Symptoms (think SLOW)
Fatigue, weight gain, cold intolerance, constipation, bradycardia, dry skin, hair loss, depression, menorrhagia
Labs
Primary (95%)
↑TSH, ↓T4
Thyroid problem
Central (rare)
↓/N TSH, ↓T4
Pituitary problem
Causes
  • • Hashimoto's - #1, anti-TPO/anti-Tg
  • • Iatrogenic (post-RAI, surgery)
  • • Iodine deficiency (worldwide)
  • • Drugs (amiodarone, lithium)
  • • Subacute thyroiditis (transient)
Treatment
Levothyroxine (T4). Start low, go slow in elderly/cardiac pts.
Myxedema Coma
Severe hypothyroidism + hypothermia + AMS + hypotension. Emergency! IV T4/T3 + steroids.

Hyperthyroidism

↑ Metabolic rate → Everything speeds up

Symptoms (think FAST)
Weight loss, heat intolerance, sweating, palpitations, tachycardia/AFib, tremor, anxiety, diarrhea, oligomenorrhea
Labs
Primary
↓TSH, ↑T4/T3
Thyroid overactive
TSH-secreting adenoma
↑TSH, ↑T4/T3
Very rare
Causes (Use RAIU to differentiate)
  • ↑ Uptake:
  • • Graves' - TSI antibodies, diffuse goiter, ophthalmopathy
  • • Toxic MNG / Toxic adenoma
  • ↓ Uptake:
  • • Thyroiditis (painful=subacute, painless=postpartum)
  • • Exogenous T4, Struma ovarii
Treatment
β-blockers (symptoms), Methimazole/PTU (synthesis), RAI, Surgery
Thyroid Storm
Severe hyperthyroidism + fever + tachycardia + AMS. Emergency! PTU + iodine + β-blocker + steroids.

Drug Effects on Thyroid

Amiodarone
Can cause HYPO or HYPER (iodine load)
Type 1: ↑synthesis in abnormal gland
Type 2: Destructive thyroiditis
Lithium
Usually causes hypothyroidism
Blocks T4/T3 release
Steroids
↓ TSH secretion, ↓T4→T3 conversion
Biotin
Interferes with immunoassays
False ↓TSH, False ↑T4 (looks like hyperthyroid)
