🔑 Key Points

  • Pattern: Cyanosis + Vigorous + No O2 response = Cardiac
  • Hyperoxia: PaO2 <100 on 100% O2 = shunting
  • PGE1: Start BEFORE echo if ductal-dependent suspected
  • Apnea: Most dangerous PGE1 side effect
  • Rashkind: Balloon septostomy creates ASD for mixing
  • Switch: Must be done within weeks while LV trained
