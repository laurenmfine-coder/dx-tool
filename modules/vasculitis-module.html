MECHANISM
Classification by Vessel Size

Vasculitis = inflammation of blood vessel walls → vessel damage, stenosis, or aneurysm. Classified by the size of the predominantly affected vessels (Chapel Hill Consensus).

CLINICAL
High-Yield Details

Giant Cell Arteritis (GCA)

⚠️ EMERGENCY: GCA can cause irreversible blindness (anterior ischemic optic neuropathy — AION). If suspected: start HIGH-DOSE STEROIDS IMMEDIATELY (do not wait for biopsy). Prednisone 60-80 mg/day. Temporal artery biopsy within 2 weeks confirms (skip lesions → need adequate specimen). Associated with polymyalgia rheumatica (PMR) in 40-60%.

ANCA-Associated Vasculitis

🔑 ANCA Patterns: c-ANCA (cytoplasmic) = anti-PR3 = GPA (Wegener). p-ANCA (perinuclear) = anti-MPO = MPA, EGPA. Mnemonic: "PR3 = P(ulmonary) R(enal) 3(GPA has 3 sites: sinuses, lungs, kidneys)".

GPA (Wegener) Classic Triad

1. Upper airway: chronic sinusitis, saddle-nose deformity, subglottic stenosis, bloody nasal discharge. 2. Lungs: nodular infiltrates, cavitary lesions, hemoptysis. 3. Kidneys: rapidly progressive glomerulonephritis (crescentic, pauci-immune on IF).

Kawasaki Disease

Diagnostic criteria (CRASH + Burn): Conjunctivitis (bilateral, non-purulent), Rash (polymorphous), Adenopathy (cervical, usually unilateral ≥1.5 cm), Strawberry tongue + lip fissures, Hand/foot changes (edema → desquamation) + fever ≥5 days. Treatment: IVIG + high-dose aspirin (one of the few pediatric aspirin indications). Echocardiogram at diagnosis, 2 weeks, and 6-8 weeks (coronary aneurysms).
