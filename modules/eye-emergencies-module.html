🏥 Clinical Correlations

Central retinal artery occlusion

Mechanism: Sudden painless vision loss - "cherry red spot"

Acute angle closure glaucoma

Mechanism: Eye pain + halos + mid-dilated pupil

🔑 Key Points

✓ CRAO: 90-minute window for treatment
✓ Acute angle closure: emergent IOP lowering
✓ Chemical burns: immediate copious irrigation
✓ Endophthalmitis: intravitreal antibiotics
✓ Always check visual acuity
