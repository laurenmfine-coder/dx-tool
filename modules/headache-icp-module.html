Primary Headache Classification
TypeFeaturesDurationTreatment
TensionBilateral, band-like, pressing. No N/V, no aura. Most common headache30 min-7 daysNSAIDs, acetaminophen. Prevention: amitriptyline
MigraineUnilateral, pulsating, moderate-severe. N/V, photophobia, phonophobia. ± Aura (visual most common — scintillating scotoma)4-72 hoursTriptans (acute, first-line), NSAIDs, anti-CGRP antibodies (prevention)
ClusterUnilateral, periorbital, excruciating ("suicide headache"). Ipsilateral autonomic signs: lacrimation, rhinorrhea, conjunctival injection, Horner syndrome. Males > females15-180 min, in clustersAcute: 100% O2 (first-line) + sumatriptan SC. Prevention: verapamil
🔑 Headache Red Flags (SNOOP): Systemic symptoms/signs, Neurologic deficits, Onset sudden (thunderclap), Older age (>50 new onset), Pattern change (progressive). Any red flag → imaging + workup.
CLINICAL
Secondary Headaches & Increased ICP

Dangerous Secondary Headaches

ConditionKey Features
SAHThunderclap headache, worst ever, meningismus. CT → LP if CT negative
MeningitisHeadache + fever + nuchal rigidity. Kernig/Brudzinski signs. LP: bacterial vs viral
Temporal arteritis (GCA)Age >50, unilateral temporal headache, jaw claudication, visual loss risk. ESR/CRP elevated. Temporal artery biopsy (gold standard). Start steroids BEFORE biopsy
Idiopathic intracranial HTN (pseudotumor cerebri)Obese young woman, headache, papilledema, pulsatile tinnitus, transient visual obscurations. Opening pressure >25 cm H₂O. Normal MRI. Drugs: tetracycline, vitamin A, OCP
⚠️ Increased ICP Signs: Headache (worse lying down/morning/coughing), papilledema, N/V, Cushing triad (HTN + bradycardia + irregular respirations = imminent herniation). CN VI palsy = false localizing sign (long intracranial course).

Herniation syndromes:
• Uncal: CN III palsy (blown pupil) + contralateral hemiparesis (ipsilateral crus cerebri compression)
• Tonsillar: cerebellar tonsils through foramen magnum → respiratory arrest
TREATMENT
Management of Increased ICP
InterventionMechanismNotes
Elevate HOB 30°Promotes venous drainageFirst step in any ICP management
Hyperventilation↓ PCO2 → cerebral vasoconstriction → rapid ICP reductionTemporary bridge only (effect wanes in hours). Target PCO2 30-35 mmHg
Mannitol (osmotic)Osmotic gradient pulls fluid from brain parenchyma20% solution. Monitor serum osmolality <320. Can cause AKI
Hypertonic saline (3% or 23.4%)Same osmotic mechanism as mannitolPreferred if hypotensive (mannitol = diuretic). No renal toxicity risk
EVD (ventriculostomy)Directly drains CSFFor hydrocephalus. Gold standard ICP monitoring
Decompressive craniectomyRemoves skull to allow brain expansionRefractory ICP elevation. Malignant MCA stroke
🔑 IIH (Pseudotumor Cerebri) Treatment: Weight loss (most effective), acetazolamide (reduces CSF production), serial LP if acute, optic nerve sheath fenestration or VP shunt if vision threatened.
