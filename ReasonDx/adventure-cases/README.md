## Features
- 3 difficulty levels (Easy, Intermediate, Advanced)
- 8 clinical reasoning questions per case
- Real-time scoring with immediate feedback
- Clinical pearls summary at completion
- Cross-linked related cases
- Mobile-responsive dark mode interface

## Usage
1. Open `adventure-hub.html` for the case browser
2. Or open any individual case directly in a browser
3. No server required - runs entirely client-side

## Integration
These cases are designed to integrate with the ReasonDx platform components:
- MechanismDx pathophysiology modules
- SpacedRep review system
- PatientUI interface
- Dashboard analytics
